How tall should the lu.ma embed be?

The embed defaults to 450px which can feel cramped with many events. It needs to be tall enough to avoid excessive scrolling inside the iframe.

A

Fixed tall height (700px)

Generous fixed height that shows several events without scrolling inside the iframe. Simple, predictable.

B

Near full-viewport (calc(100vh - 200px))

Embed fills most of the screen below the header/title area. Feels like a native page rather than an embedded widget. Responsive to screen size.

C

Keep lu.ma's default (450px)

Use the iframe as-is. Compact, but users will scroll inside the iframe a lot if there are many events.
