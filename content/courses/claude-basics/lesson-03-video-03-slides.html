Claude's Memory: Two Rules

✅

Within a Conversation

Claude remembers everything. Every prompt, every response, every detail you've shared. This is why iteration works so well.

❌

Across Conversations

Claude starts with a blank slate. Each new conversation is like meeting Claude for the first time. No memory of yesterday.

One conversation = Full memory. New conversation = Fresh start.

Section 2 • 3:00

Your Conversation Shortcuts

Point Claude back to earlier parts of the conversation:

"In your last answer, you mentioned..."
"Based on what you suggested above..."
"Using the same approach as before..."
"Can you apply that same logic to..."
"The code you wrote for the header..."
"You mentioned X, can you go deeper?"

These save time — no need to repeat context Claude already has.

Section 3 • 5:30

The 10-Second Evaluation

1
Does this answer my question?
No → Rephrase
2
Is it the right level of detail?
No → Expand / Constrain
3
Is the format useful?
No → Restructure
4
Would I use this as-is?
Yes → Done!

This takes 10 seconds but saves minutes of aimless iteration.

Section 4 • 7:15

Three Signs You're Done

1

Changes are cosmetic, not substantive

You're tweaking word choices, not fixing real problems.

2

You're going back and forth

You change something, then change it back. You were happy two iterations ago.

3

You read it and think "I'd use this"

Trust that feeling. If it answers your question and you'd use it, stop iterating.

5+ iterations with no progress? Start fresh with a better prompt.
