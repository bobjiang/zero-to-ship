Section 2 - 3:30

How to Reference Effectively

OK (if only one)

"The auth system we built"

Works if you have one auth conversation

Better (always clear)

"The login form with email validation from our user authentication conversation"

Specific enough to find the exact context

The more conversations you have, the more specific you should be.

Section 3 - 5:30

Continue or Start New?

Should I continue this conversation?
↓
↓

Continue

  • Same feature in progress
  • Conversation still manageable
  • Building on previous responses

Start New

  • Different feature or topic
  • Conversation got very long
  • Want a fresh approach

Think of each conversation like a chapter in a book. New topic = new chapter.
