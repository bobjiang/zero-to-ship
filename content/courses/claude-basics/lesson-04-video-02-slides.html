Section 2 - 3:45

Installing Node.js

Go to nodejs.org and pick the LTS version (Long-Term Support = stable and tested).

Download, run installer, accept defaults. One-time install -- then forget about it.

Always choose LTS (Long-Term Support)

Verify Installation
$ node -v
v20.11.0

$ npm -v
10.2.4

// Both show versions? You're set!
Section 3 - 5:30

package.json

Your project's recipe card. Lists the project name, commands, and dependencies (external libraries).

Claude creates this for you. You just need to know it exists.

It tells npm what to download.

{ "name": "my-project", "scripts": { "dev": "next dev", "build": "next build", "start": "next start" }, "dependencies": { "next": "^14.0.0", "react": "^18.2.0", "react-dom": "^18.2.0" } }

Section 5 - 12:00

Errors Are Clues, Not Dead Ends

"node is not recognized"
→
Reinstall Node.js
"port 3000 already in use"
→
Close other servers or use different port
"module not found"
→
Run npm install again

Best trick: Copy the error, paste it to Claude, ask for the fix.
