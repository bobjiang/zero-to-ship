Section 3 - 7:00

Knowledge Base: What to Upload

Upload These

  • ✓ README or project overview
  • ✓ Requirements documents
  • ✓ Key code samples (layouts, models)
  • ✓ Style guides or design references
  • ✓ Type definitions

Skip These

  • ✗ Entire codebase
  • ✗ API keys or passwords
  • ✗ .env files
  • ✗ Massive log files
  • ✗ Generated/temporary files

Quality over quantity. Upload key docs, not everything.
