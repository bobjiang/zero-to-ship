Section 3 - 5:00

The Golden Rule of AI Code

Read Before You Run

You don't need to understand every line. Just look for things that seem off: file deletions, unknown network requests, or password prompts. When in doubt, ask Claude to explain.

Section 3 - 6:00

Protect Your API Keys

API keys are like passwords for services. If someone gets yours, they can use your account and charge you money.

If Claude ever asks for your password, something is wrong. Claude doesn't need your passwords to help you.

Section 3 - 6:45

Beginner Code Review Checklist

You don't need to be a security expert. Just use common sense.

1 Does Claude's explanation match what you asked for?
2 Are there file operations you didn't request?
3 Does the code reference websites or services you didn't mention?
4 When in doubt: "Can you explain what this code does step by step?"
